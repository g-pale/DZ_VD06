## Описание проекта

Этот проект — простое веб‑приложение на **Flask**, выполняющее задание:

- **Создать форму анкеты пользователя** с полями: имя, город, хобби и возраст.
- **Использовать фреймворк Bootstrap** для оформления.
- **После отправки формы показывать введённые данные на той же странице**.

Все требования задания выполнены в шаблоне `forma.html` и маршруте `index` в `app/routs.py`.

## Структура проекта

- `main.py` — точка входа, запуск Flask‑приложения.
- `app/__init__.py` — создание объекта приложения `app`.
- `app/routs.py` — маршрут `/`, обработка формы и передача данных в шаблон.
- `app/templates/forma.html` — HTML‑шаблон с формой и таблицей результатов (Bootstrap).

## Логика работы

1. При открытии корневого URL (`/`) обрабатывается маршрут `index` из `app/routs.py`.
2. При **GET‑запросе** отображается форма анкеты (шаблон `forma.html`).
3. При **POST‑запросе**:
   - Читаются поля формы: `name`, `city`, `hobby`, `age`.
   - Если все поля заполнены, данные добавляются в список `posts`.
   - Выполняется `redirect` на `index`, чтобы страница перезагрузилась.
4. В шаблон `forma.html` передаётся список `posts`, и внизу страницы выводится таблица всех введённых анкет.

## Использование Bootstrap

В `forma.html` подключён **Bootstrap 5** через CDN:

- Оформление страницы (`container`, `row`, `card`, `table` и т. д.).
- Адаптивная вёрстка и приятный внешний вид формы и таблицы.

## Как запустить проект

1. Создать и активировать виртуальное окружение (по желанию).
2. Установить зависимости (минимально нужен Flask):

```bash
pip install flask
```

3. Запустить приложение:

```bash
python main.py
```

4. Открыть в браузере:

```text
http://127.0.0.1:5000/
```

5. Заполнить форму и отправить — введённые данные появятся в таблице под формой на той же странице.
